# Scaling Laws for Neural Language Models

## 초록
- The relationship between overfitting and model/data size, as well as between training speed and model size, can be expressed with simple equation

## 요약
- 모형의 성능에 영향을 크게 미치는 3요소
1. N : Model's parameter size
2. D : Dataset size
3. C : 학습에 사용된 연산량
- 극단적인 경우를 제외하고는 모혀으이 성능은 깊이와 너비의 상대적으로 약한 영향을 받음

- N, D, C는 멱 법칙을 따름(병목 현상이 일어나지 않는다는 가정)
- 멱법칙 : $y=ax^k$. 작은 규모의 사건은 빈번하고, 큰 규모의 사건은 드물게 발생
- N, D를 늘리면 성능 향상. 하나를 고정하고 다른 하나를 늘리면 수확 체감의 법칙(처음에는 성과가가 잘 늘다가, 어느 순간부터는 증가폭이 점차 줄어듬)
- If the model size increases by 8x, this can be avoided if the dataset is increased by approximately 5x
- 학습 곡선은 멱 법칙을 따름
- 외삽법을 초기 학습 곡선에 도입함으로써 손실을 예측 가능
- 연산량이 고정일 때, N과 D에 제한이 없다면 매우 큰 모형은 학습을 수행할 때 수렴하지 않음에도 조기 종료를 적용함으로써 최적에 도달 가능
- The amount of data required for optimal training is smaller than expected, foloowing approximately. $ D \sim C^{0.27}$
- 최적의 배치 크기는 손실에 대한 멱 법칙을 따름. It can be determined by measuring the gradient noise scale
