# InstructGPT: Training language models to follow instructions with human feedback

## 1. 서론
- While performance on public NLP datasets may improve, it doesn't necessarily capture human intent accurately
- Just train predicting next token => Misaligned
- Use RLHF(Reinforcement Learning from Human Feedback)
- Supervised fine-tuning(SFT) + Reward Model(RM)

## 2. 방법

### Dataste
1. Prompt dataset made by labeler
2. A dataset consisting of prompts generated by users interacting with existing models

- Prompt example
  - 점심을 추천해줘
- 3 types prompt data
  1. Plain : 여러 주제에 대한 자유로운 작성
  2. Few-Shot : 지시 형식으로 작성, 그에 관한 몇 가지 예시 같이 작성
  3. User-Based : Developed by referencing prompts created by users

### 모형
- 세 가지
  1. SFT
  2. RM
  3. PPO(Proximal Policy Optimization)

1. SFT
  - 지도학습을 통한 미세 조정
2. RM
  - 모형이 반환하는 여러 출력에 대해 순위를 매김
  - 순위를 바탕으로 학습.
  - 선호하는 출력에 대해서는 보상을 많이 주는 방식으로 학습
  - 인간의 선호를 잘 반영한 모형
3. PPO
  - 정책 자체를 최적화시키는 방법
  - 강화 학습과 지도 학습 결과가 너무 차이나는 것을 방지
  - 사전 학습된 분포를 잊지 않기 위해 PPO-ptx 사용
