# Training Compute-Optimal Large Language Model

## 초록
- The number of training tokens and the model parameter size should be scaled up at the same rate
- $N_{opt}(C), D_{opt} (C) = argmin L(N,D)$
- Their goal is to find the optima number parameters and training tokens that can be allocated under a given compute budget

## 관련 연구

### 멱 법칙(Power-Law relationship)
- 가중치 수와 성능 사잉의 연관관계
- $Loss \propto N^{-\alpha}$
